yt_live_scraper.scraper.is_stream_live(video_id, *, timeout=10)

Check whether a YouTube stream is currently live.

Fetches the video page and inspects the player response for the isLive flag inside videoDetails.

Parameters:

  • video_id (str) – YouTube video ID to check.

  • timeout (float) – HTTP request timeout in seconds.

Return type:

bool

Returns:

True if the stream is currently live, False otherwise (including on network errors).
