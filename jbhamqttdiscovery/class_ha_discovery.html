Detailed Description

Home Assistant MQTT Discovery publisher (transport-agnostic).

Typical usage:

  • Construct with a MqttTransport implementation
  • Provide device info via setDevice()
  • Publish discovery configs (retained)
  • Publish states as needed

For async transports, onConnect is invoked automatically. For sync transports, call tick() periodically to detect reconnection transitions.
